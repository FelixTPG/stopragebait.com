# 💥 StopRagebait.com

StopRageBait is a web app designed to help users identify, filter, and avoid rage-bait content online. It aims to be a fun site by showing how to avoid ragebait online with a bit of sarcasm.

## 🌐 Live Site
https://stopragebait.com

---

## 🤔 What is Rage Bait?
Rage bait is content intentionally designed to provoke anger or outrage to drive engagement. StopRageBait helps users recognize patterns and take control of their feed.

## 🚀 Getting Started

### Prerequisites

* Node.js (>= 18 recommended)
* npm or yarn

### Installation

```bash
git clone https://github.com/FelixTPG/stopragebait.git
cd stopragebait
npm install
```

### Development

```bash
npm start
```

Runs the app in development mode.

### Build

```bash
npm run build
```

Builds the app for production.
